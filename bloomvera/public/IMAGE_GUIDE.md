# Images Required in /public folder

Place these image files in this /public directory before running the app:

## Logo
- logo.png — Your Bloomvera Autism logo (used in Navbar + Footer)

## Hero Images (Home page)
- hero-1.png — Main hero image (child in therapy / happy child)
- hero-2.png — Secondary hero image (therapist with child)

## About Page
- about-1.png — Team photo or therapist working (landscape ratio)
- about-2.png — Child with therapist or program image (landscape ratio)

## Services Page (one per service)
- aba-therapy.png — ABA therapy session image
- social-skills.png — Group session / social skills image
- daily-skills.png — Child learning daily skills
- home-support.png — Home-based therapy session

## Recommended image sizes
- Logo: 200x80px (PNG with transparent background)
- Hero images: 800x600px or larger, landscape
- About images: 800x500px, landscape
- Service images: 800x450px, landscape (16:9 ratio)

All images gracefully fall back to styled placeholder sections if not found.
